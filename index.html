Profesional Skills

HTML-5 is a markup language most commonly used to create and present websites on the World Wide Web, with it you are able to create and organize diferent aspects and parts of a website, for example; a footer that contains the contact information, a form that allows the user to send a meessage to the server, a table of content that displays different offers that the user can then select, a navigation menu that allows the user to quickly find the different sections in a website, and many other aspects that are necessary in a good website.

CSS stands for Cascading Style Sheets, and is normally used for changing and modifing the style of a document written in a markup language such as HTML, an example would be the desing and the aesthetics of a website can be modified using CSS. Another important aspect of CSS is that it allows us to create a responsive website by changing and modifing the styling of the website depending on the size of the device being used to view the website.

SCSS (or Sass) is basically an advance version of CSS but with more features that can make coding much simpler, one of these features is mixins, mixins allow you to shorten some parts of the code where instead of writing the same 4 or 5 lines of code over and over again in CSS you can simply write those lines one time and after that you only need to insert 1 line in your code whenever you need to apply that mixin, reducing the size of your code and the amount of work you have to do.

Flexbox is used in CSS to arange the layout of elements within a container and is commonly used on responsive websites because it makes designing and building responsive websites much simpler by reducing or even removing the need for a lot of float and position properties in the CSS code.

A responsive website is one that changes and adapts depending on the size of the device that is being used to see that website, for example the 3 most common versions that are taken into consideration when developing a responsive website are the mobile version which normaly has a minimum width of 300px, the tablet version usually has a minimum width of 700px and the desktop versions ususally has a minimum width of 1000px.
